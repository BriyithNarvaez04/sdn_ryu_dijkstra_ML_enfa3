Detailed Description

Script de emulación en Mininet para la topología de la red troncal europea GÉANT.

Configura un entorno SDN (Software-Defined Networking) compuesto por 23 switches OpenFlow y 23 hosts con el protocolo STP habilitado. Orquesta un pipeline automatizado de generación de tráfico HTTP intermitente para la recolección de muestras, simulando patrones bimodales (ráfaga/reposo) con el fin de alimentar y validar el modelo clasificador de Machine Learning.
